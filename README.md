# 🛡️ Polio Coverage Monitoring Portal

## Overview

The Polio Coverage Monitoring Portal (PCMP) is a modern healthcare dashboard designed to track and monitor polio vaccination campaigns across cities, regions, and micro-areas. The platform provides interactive data visualization, geographic mapping, team monitoring, and reporting capabilities to support vaccination coverage analysis.

## Features

### 📊 Dashboard Analytics

* Campaign overview and key performance indicators
* Vaccination coverage statistics
* Coverage rate monitoring
* Monthly performance trend analysis

### 🗺️ Geographic Coverage Mapping

* Interactive maps powered by Leaflet.js
* Coverage visualization across different regions
* Color-coded status indicators
* Geographic campaign monitoring

### 🏙️ City Management

* City-wise vaccination coverage tracking
* Coverage percentage monitoring
* Search and filtering functionality
* Status-based performance indicators

### 📍 Area Tracking

* Micro-area level monitoring
* Coverage percentage calculations
* Vaccinated population tracking
* Status filtering and analysis

### 👥 Vaccination Team Management

* Field team deployment overview
* Daily target tracking
* Progress monitoring
* Team status management

### 📈 Reports & Analytics

* Coverage ratio charts
* Performance trend visualization
* Reporting interface
* Data export workflow prototype

### 🌙 User Experience

* Responsive design
* Dark mode support
* Mobile-friendly interface
* Modern dashboard layout
